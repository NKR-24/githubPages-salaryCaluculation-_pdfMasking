# 明細書マスキング v2.1（テキスト保持型）

保険手数料明細PDFの**契約者名と会社名**をブラウザ内で墨消しするツールです。

**公開URL:** https://nkr-24.github.io/githubPages_PrimeOne_-pdfMasking/

v2 は「真の墨消し」方式です。黒塗り箇所は見た目を隠すのではなく**文字データごとPDFから削除**します。一方、それ以外のテキスト層（証券番号・金額・ページ左上のコードなど）はそのまま保持されるため、出力PDFは検索・コピー・AI読み取り（Excel化）がOCRなしで行えます。

## 対応様式

| 欄 | 様式（1ページ目の文字列で自動判定） | 墨消し対象 |
|---|---|---|
| A | あいおい生命「手数料支払明細書」 | 契約者名（カナ行含む） |
| B | 三井住友海上「手数料早期支払明細書」 | 契約者名の列（内容不問・全行）＋会社名の行 |

間違った欄にアップした場合は、ファイルを変更せずエラーを表示します。

## 使い方

1. 公開URLを開く（初回のみエンジン約10MBをダウンロード。以降はキャッシュ）
2. 該当する欄（A / B）にPDFをドラッグまたはクリックして選択（複数ファイル可）
3. ファイルごとの**検証カード**を確認
   - ① 削除した文字列の残存 … 0件
   - ② 黒塗り領域内の残存文字 … 0件
   - ③ 先頭3ページのプレビューで黒塗り位置を目視
   - 3点合格で「検証済」スタンプが付きます
4. 「〇〇_masked.pdf をダウンロード」で保存（2件以上あれば「すべて保存」も可）
5. アップ欄の下の**Excel化用プロンプト**（A/B別）を「コピー」し、ダウンロードした masked PDF と一緒にClaudeへ貼り付ければ後工程がそのまま始められます（文面は編集可）

パスワード付きPDFは、先にロック解除してからアップしてください。

## v1 からの変更点

- v1: ページ全体を画像化して黒塗り → 文字情報が全消失 → 後工程（Excel化）で全ページOCRが必要で低速
- v2: 該当文字だけを削除する真の墨消し → テキスト層保持 → **OCR不要でAI処理が高速**
- ファイルごとの自動残存チェック＋プレビュー（検証カード）を追加
- Excel化用プロンプトのコピー欄（A/B別）を追加
- 手動黒塗りモードは廃止（必要になれば再追加可能）

### v2.1 での修正
- ダウンロードしたPDFが開けない不具合を修正（保存データをWASMメモリからコピーせず参照していたため、検証・プレビュー処理後に中身が失われていた）
- 合計ページで金額などを契約者名と誤認して「残存あり」と警告する誤検知を修正
- 注意書き・エラー警告文の表示を削除（検証結果の✓/✗表示は継続）

## 座標がズレたとき

様式変更などで黒塗り位置がズレた場合は `redactor.js` 冒頭の `FORMATS` の数値を調整します（単位: PDFポイント、ページ左上が原点）。

- **aioi（あいおい生命）**: `anchorX: 123.8`（データ行の左端）、`nameCharIndex: 5`（契約者名は行の6文字目から）、`nameX0: 159.6`（名前列の左端）、右端は「口振」ヘッダの左端 −0.5（見つからないときは `fallbackNameX1: 238.5`）、`skipWords`（小計・契約者名の行は対象外）
- **ms（三井住友海上）**: `band: [140.0, 213.5]`（契約者名列の左右端）、`bodyY: [152.0, 584.0]`（表本体の上下端。ヘッダ・フッタを除外）

## 技術メモ・制約

- 処理はすべてブラウザ内で完結し、ファイルがサーバへ送信されることはありません
- ES module + WebAssembly を使うため **file://（ダブルクリック）では動きません**。公開URLで開くか、ローカル確認は `python -m http.server` → http://localhost:8000
- `lib/` の読み込みに失敗した場合は jsDelivr CDN から自動フォールバックします
- 動作環境: WebAssembly対応の最新ブラウザ（Chrome / Edge / Firefox / Safari）。大きいファイルはPC推奨
- 更新方法: **Add file → Upload files** に `index.html`・`redactor.js`・`lib` フォルダをドラッグして上書き → Commit → 1〜2分後に Ctrl+F5
